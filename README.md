# Are-cortical-microcircuits-optimized-for-information-flux-
The complete simulation of the paper with same name
https://arxiv.org/abs/2605.14680

The code may be a bit confusing, which is understandable! Documentation has been created with the help of AI to specify each class's functionality in the whole paper. To add to this, I created the GOAL PDFs to summarize all the figures of the said paper, and based on which goal everything has been done. Also, for the code to be understandable, I added extra comments in each file with the help of AI to describe the work in each file, the loops and lines, and what they are doing. In case of a problem, please contact me, and I'll help you.
